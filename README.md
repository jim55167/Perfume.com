## 簡介
此作品為 Vue.js 前端練習，功能包含：
+ 前台
  + 電商展示（商品分類、單一商品介紹）
  + 會員登入(顯示/隱藏密碼)
  + 購物車
  + 優惠碼
  + 下單/結帳
+ 後台
  + 管理者登入
  + 商品管理（建立、更新、刪除）
  + 優惠碼管理（建立、更新、刪除）
  + 訂單列表
  
## 使用技術
